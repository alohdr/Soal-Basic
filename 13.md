# Array to String

Buatlah script yang dapat mengubah array-object menjadi string

```js
const fruits = [
  {name: 'Pepaya', price: 3000},
  {name: 'Jagung', price: 2000},
  {name: 'Mengkudu', price: 4000},
  {name: 'Kiwi', price: 6000},
]
```

### Contoh Output

```
Pepaya: 3000
Jagung: 2000
Mengkudu: 4000
Kiwi: 6000
```
